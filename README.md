# EscapeRoomGameProject
Escape room game Development Project (C, 2021)

This is a simple escape room game c language code consisting of header code and main execution code. 

It consists of a total of 3 steps, 
1st step is to escape the room by acquiring the key, 
2nd step is to escape the room by avoiding obstacles and acquiring the key, and 
3rd step is to escape the room by acquiring the key in the maze.
